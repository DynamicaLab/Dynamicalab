Datasets¶

Usage¶

We have created a convenient dataset loader. The datasets are automatically downloaded in the save_path. If the data have already been downloaded, the dataset are automatically loaded from the local path.

Each dataset has a graph generator method to convert the raw data to a nx.Graph.

Example¶

import dynamicalab as dlb

data = dlb.plants_pollinators_McCullen1993(save_path="./datas")
G = data.graph()
