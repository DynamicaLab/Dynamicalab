dynamicalab.plants_pollinators_Herrera1988¶

dynamicalab.plants_pollinators_Herrera1988(save_path='./data')[source]¶

Plant pollinators dataset of Herrara 1988 .

  • Nodes: 205 (26 plants, 179 pollinators)
  • Edges: 412
  • Bipartite: True
  • Components: 1

Reference
